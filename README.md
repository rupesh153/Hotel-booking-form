Hotel Booking Form
 Project Description

The Hotel Booking Form is a responsive web-based form designed to allow users to book hotel rooms by selecting check-in and check-out dates, room type, number of guests, and adding special requests.
This project focuses on implementing proper form validation, date validation logic, responsive design, and dynamic form behavior using HTML, CSS, and JavaScript.


🛠️ Technologies Used

HTML5 – Structure of the form

CSS3 – Styling and responsive design

JavaScript (Vanilla JS) – Form validation and dynamic behavior

Git & GitHub – Version control
